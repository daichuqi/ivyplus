    简介

    八十年代的创业者们还在政策的夹缝中寻找机遇，在僵化的计划经济体制日渐瓦解的情况下，一群有着龙虎之志的“小人物”在中国这个大试验场里面开辟出了一番前所未有的局面。九十年代的房地产市场，巨大的诱惑让体制内的精英们纷纷出走，去外面的世界积累财富。外资的进入，为中国的企业提供了更先进多元的理念。愈加开放的市场，也让企业家们有了更广阔的眼界。千禧年之后，伴随经济全球化以及软件行业的发展，创业也不再是只有一小部分人可以去实践的工作。而如今，互联网时代的到来，让每一个年轻人都有能够实现改变世界梦想的可能。青年海归在这场互联网时代的巨大变革中扮演的重要角色是任何人不容忽视的。

    这是一个最好的时代，我们的生活正因为各种海归青年创业者的努力变得更加有趣而高效。这是一个最好的时代，海归年轻人在享受科技和创新带来的快乐和幸福时候，国内的政策和融资环境也为他们提供了去创造和改变的可能。常春藤名校在美国发展史乃至世界发展史上都输出了极多的关键性人才，在创业领域，泛常青藤名校学生也表现尤为突出，从老一辈的微软创始人、世界首富Bill Gates，到当今炙手可热的科学狂人Elon Musk，再到一号店创始人于刚，都是常青藤校友。

    Ivy Plus 创业大赛旨在建造互联网时代提供链接国内投资者、孵化器、媒体、创业园区、成功创业者和年轻的泛常青藤名校精英留学生创业者供需匹配的桥梁。大赛将搭建海外留学人员创新创业日常展示交流、资源对接、项目孵化等平台，整合创业服务机构、创业园区、创投基金、创业导师等资源，遴选优质创业项目，协助项目落地，提升创业成功率。

    主办方
    常青藤中国论坛

    “常青藤中国论坛”成立于2012年，是由常青藤院校的中国师生们自发组织的中国峰会。它旨在为留学生群体与中美两国的政商学三界领袖提供深入交流的平台，以帮助双方进行具体、实际的讨论，是选择归国的留学人才于国内各个行业的需求达到最高效、完美的对接。

    Read More

    2013年至今，常青藤中国论坛已举办三届，场场爆满，嘉宾包括杨振宁、施一公、俞敏洪、徐小平、熊晓鸽、汪潮涌、阎焱、王志东、许小年等各界精英。获得了包括新华社、中青报、中国企业家杂志、IDG资本等多家媒体和机构的支持，常青藤联盟各校学生学者联合会都作为校园合作方支持论坛。其中，2015年的第三届常青藤中国论坛联手著名科技媒体36氪，聚焦当下最热点的互联网创业，邀请到王功权、李丰、雕爷、袁岳等著名创投人士，汇集业内资深人士与新生代创业明星，全方位多角度把握互联网命脉，探索未来互联网发展趋势，为海归创业指点迷津。常青藤论坛作为国内最具高端和领先的品牌效应、知名度最高的留学生平台，在创投领域和留学生内都有极高的影响力。

    南京市白下高新技术产业园区

    园区位于秦淮区东部,地处紫金山、青龙山的生态绿廊之间，毗邻运粮河百里风光带，总占地2.1平方公里，是南京市主城区中唯一的省级开发区。园区周边高校、科研机构资源十分丰富，聚集南京理工大学、南京航空航天大学、解放军理工大学、南京农业大学等13家高等院校，28所、55所、8511所和63所等37家科研机构。

    Read More

    园区坚持“以人才带动产业，以产业集聚人才”战略部署，着眼产业高端定位，主要发展云计算、智能交通、电子商务、生物医药、新材料、文化创意、节能环保等高新技术产业。2011年起，园区全面启动招才引智工作，共集聚南京“321”人才121人（65人有海外背景）。其中南京市领军型科技创业人才115人，南京市科技创业家6人，江苏省“双创”11人，国家千人计划11人，院士1人。园区人才321申报通过率及入选人才总数均名列前茅。

    围绕科技人才企业专业化、多元化需求，创新推出“全程覆盖、按需服务”理念。目前“1+X”企业服务中心也已经投入使用，对人才创业整个过程实现了全面覆盖。 服务内容包括：为孵化企业提供市场拓展服务；孵化项目提供项目经理以及24小时的随叫随到服务；与南京的各科技银行合作，为特区企业提供“见保即贷”的优质待遇等等。通过各项服务，目前有订单人才企业60家，占园区人才企业总数的48%，累计实现销售收入（含订单）18.9亿元。

    36氪(氪空间)

    36氪创立于2010年，是中国最大的创业生态服务平台，在创业服务领域已经是绝对的行业领导者。36氪已形成三条支柱业务线：36氪融资、氪空间与36氪媒体。

    Read More

    36氪融资平台，已包括31000多家创业公司，总量行业第一。2015年上半年帮助了近100家企业完成融资。仅2015年6月15日当周，36氪融资平台帮助创业者、投资人完成对接4051次，创业者和投资人的相遇从线下转移到线上，目前同时在36氪融资平台挂牌融资的企业达到3100多家，36氪融资平台几乎垄断了整个TMT创业融资领域，绝对的第一。36氪股权投资平台（原名：36氪股权众筹平台）如今已经成为互联网业内最大的股权融资平台，截止9月7日，已经完成融资额5000万人民币。未来3年，36氪融资平台将为10,000多家企业提供融资服务。

    36氪线下孵化器--氪空间过去一年累计孵化70多家企业，这些企业的融资成功率接近100%，是中关村第一批“创新型孵化器”的典型代表，也得到了国务院、科技部等领导的高度赞赏。现在，氪空间已开启了全国化之路，截止目前，已经在全国6大创业城市入驻，杭州将成为首个正式落地氪空间的城市。氪空间将在接下来的几年完成每年孵化5000家企业的目标，品质和数量都将成为行业绝对第一。

    协办方: 峰瑞资本（FreeS VC)、麻省理工学院中国创新与创业论坛（MIT-CHIEF）、2015香港国际创客节

    FAQ

    本届大赛开启报名以来，受到了来自社会各界的密切关注，我们同时也发现粉丝们存在着一些普遍对于大赛的问题，我们将最常见的问题总结如下：

    Q1: 创业项目的方向是否有限制？是否一定要互联网创业？
    现在绝大部分的创业都和互联网相关，当然本届大赛并不局限于纯互联网创业，硬件、新技术当然无限欢迎。
    Q2: 我们是一个大学生团队，我们有一个很好的想法，但是团队都没有工作经验，是否可以报名参加？
    当然可以，我们欢迎所有背景的创业团队，年轻人往往更有创造力！
    Q3: 我们这个项目目前只是一个idea，并没有产品是否可以参加本次大赛？
    当然可以参与，我们非常期待被创新和革命性的idea洗礼。
    Q4: 第一次看到这样的创业大赛，不知道你们的专业性如何？
    我们有超过30家最顶级的VC支持并且所有主办方都是业内最优秀的单位与机构，相应的获奖奖励政策也是真材实料。所以专业性会得到充分的保障。
    Q5: 请问本次大赛是否有设立奖金？
    本届大赛并未设立奖金，因为我们清楚地了解，真正对创业者有帮助的是与顶级VC面对面的机会以及来自于行业内大拿的指导。不设立奖学金是为了吸引真正的创业者参赛。但同时，优胜团队将有机会获得321人才工程提供的100万元到200万元创业扶持基金。
